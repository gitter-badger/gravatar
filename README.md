gravatar
========

Simple class to generate Gravatar URL for a email-address.

Usage:
------

```java
import org.rjung.util.Gravatar;

class GravatarExample {
  public void main(String[] args) {
    System.out.println(Gravatar.getInstance().imageUrl(
        "example@example.com"));
  }
}
```
